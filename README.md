# ai-notes-frontend
#  AI Notes Generator (Frontend)

A modern frontend application that converts **PDFs or text into structured notes using AI**.
Built with **React.js**, this app provides a smooth and interactive UI for generating, viewing, and managing notes.

---

## ✨ Features

*  Upload PDF and generate notes instantly
*  Convert raw text into structured notes
*  Fast AI-powered processing
*  Clean and readable note formatting
*  Markdown support for rich content
*  Copy notes with one click
*  Fully responsive design

---

## 🛠️ Tech Stack

* **React.js** – UI development
* **Tailwind CSS** – Styling
* **Axios** – API communication
* **React Markdown** – Render notes
* **React Router DOM** – Navigation

## ⚙️ Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/your-repo.git
```

### 2️⃣ Go to Frontend Folder

```bash
cd frontend
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Setup Environment Variables

Create a `.env` file:

```env
VITE_API_URL=http://localhost:5000
```

---

### 5️⃣ Run the App

```bash
npm run dev
```
